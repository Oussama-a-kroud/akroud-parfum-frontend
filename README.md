# 💎 Akroud Parfum - Frontend (E-commerce)

A modern, fast, and responsive e-commerce web application for a perfume store, built with React and Tailwind CSS.

## 🚀 Features
* **Modern UI/UX:** Mobile-first, fully responsive design adapted for all devices.
* **Dynamic Catalog:** Real-time display of perfumes with different variants (sizes and prices).
* **Interactive Cart:** Seamless and real-time shopping cart management.
* **Smart Checkout (WhatsApp):** Optimized and automatic redirection to WhatsApp with a complete order summary. Built to bypass iOS and Android pop-up blockers effectively.

## 🛠️ Technologies Used
* **Frontend:** React.js, Vite
* **Styling:** Tailwind CSS
* **HTTP Requests:** Axios
* **Notifications:** React Hot Toast
